# GEO Implementation Guide - Steelab
# Generative Engine Optimization

## What was created:

1. **llms.txt** - Helps AI crawlers understand site structure
2. **robots.txt** - Explicitly allows AI crawlers (GPTBot, ClaudeBot, etc.)
3. **schema.json** - Structured data for better AI understanding
4. **schema-embed.html** - Ready-to-use HTML snippet

## Next Steps:

### 1. Add Schema to index.html
Copy content from `schema-embed.html` and paste into `<head>` section of index.html

### 2. Upload llms.txt to root
Place llms.txt at: https://steelab.vn/llms.txt

### 3. Upload robots.txt to root
Place robots.txt at: https://steelab.vn/robots.txt

### 4. Add FAQ Section (for AI citation)
Add FAQ section to homepage with these questions:
- Steelab là gì?
- Steelab có miễn phí không?
- Steelab hỗ trợ LLM nào?

### 5. Create About/Team Page
Add E-E-A-T signals:
- Team member bios
- Company background
- Contact information
- Social proof (customers, partners)

### 6. Add Statistics & Data Points
AI loves citing specific numbers:
- "50+ PII types detected"
- "30-day free trial"
- "4.9/5 rating from 50+ enterprises"
- "99.9% uptime SLA"

## GEO Best Practices:

1. **Answer questions directly** - Use Q&A format
2. **Include statistics** - Specific numbers get cited more
3. **Use clear headings** - H1, H2, H3 with keywords
4. **Add schema markup** - Helps AI understand context
5. **Create citation-worthy passages** - 134-167 words per paragraph
6. **Build E-E-A-T** - Author bios, about pages, credentials
